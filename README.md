# Odin Recipes

A simple recipe website built for The Odin Project Foundations course.

## Features
- Homepage with links to recipes
- 3 recipe pages with ingredients and instructions
- Images for each recipe
- Proper HTML structure

## Skills Demonstrated
- HTML basics (headings, paragraphs, lists, images, links)
- Git workflow (clone, add, commit, push)
- File organization and directory structure
- GitHub repository management
